## Computer Business Management System  

**Technologies Used:** Oracle MySQL  

### Project Overview  
This project is a comprehensive database system designed to manage various aspects of a computer business. It centralizes data related to owners, products, branches, employees, and customers, ensuring efficient and organized business operations.  

### Key Features  
- Centralized database for business management  
- Management of owners, products, branches, employees, and customers  
- Efficient data organization using relational database design  
- Supports tracking of product inventory, pricing, and sales  

### Database Structure  
The system stores and manages the following entities:  

- **Owners:** ID (Primary Key), Name, Phone (Multi-valued), Address  
- **Products:** Product Number (Primary Key), Model, Quantity, Purchase Price, Selling Price  
- **Branches:** Branch Number (Primary Key), Location, Telephone  
- **Employees:** ID (Primary Key), Name, Phone (Multi-valued), Salary  
- **Customers:** Name, Invoice ID, Address  

### Purpose  
The goal of this project is to streamline computer business operations through a structured and scalable database system, improving data management and accessibility.
